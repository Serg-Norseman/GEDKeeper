Development

Short list of Features:

  • simple and intuitive interface
  • easy install
  • high compatibility with GEDCOM 5.5.1 format
  • partial compatibility with GEDCOM 5.5EL format
  • no restrictions on the number of persons and generations
  • generation of pedigrees
  • rendering several trees diagram kinds
  • print and export tree diagrams as image
  • fast navigation, filtering and searching
  • adding multimedia materials
  • output of geographic data on maps (Google, Yandex, OSM)
  • cross-platform project (Windows, Linux)
  • supports Lua scripts and plugins
  • files password protection
  • pedigree reports
  • possibility to export PDF, Excel, RTF, HTML
  • statistics
  • organizer and slideshow
  • reminder of birthdays
  • support of multi-monitor configurations
  • calculator of relationship degree
  • reminder of holidays
