Родословные росписи

В программе GEDKeeper реализована возможность генерации поколенных родословных росписей. Росписи генерируются двух видов: по системе д'Абовилля и по системе Коновалова. Собственно роспись в обоих случаях одинакова, отличается лишь нумерация персон.

Согласно системе д'Абовилля первый человек в первом поколении получает номер 1, первый его потомок в следующем поколении 1.1, внук - 1.1.1 и т.д. Система удобна в том случае, если древо ведется изначально на бумаге - легко добавлять элементы в любое поколение росписи.

Согласно системе Коновалова все люди в росписи получают номер вида "X-Y", где X - порядковый номер человека в росписи, а Y - номер его предка. В этой системе очень легка навигация, однако при ведении на бумаге затруднено добавление новых персон в уже сформированные поколения - нарушается порядок цифр, роспись становится хаотичной.

Для программы GEDKeeper разница между этими двумя системами несущественна и росписи можно формировать в любой без ограничений. Значение имеет лишь предпочтение пользователя к внешнему виду росписи и личному удобству при просмотре и навигации.

Для построения родословной росписи нужного вида, необходимо выбрать в списке персон интересующего человека, а затем в меню "Родословная" выбрать пункт нужной системы. Также можно нажать кнопку на панели инструментов.

Существует несколько форматов файлов, в которые можно экспортировать родословные: HTML, RTF и PDF. Экспортированный файл содержит гиперссылки между персонами в дереве. Cозданный файл сразу открывается в программе, которая зарегистрирована в системе в качестве действия по умолчанию для указанного типа файлов. Вы можете использовать такие программы, например, для печати родословных.

Настройки генерации родословной росписи располагаются в диалоге "Настройки", закладка "Росписи".

  • Настройка "Включая атрибуты персон" - задаёт то, будут ли включаться в роспись факты из жизни человека.
  • Настройка "Включая заметки" - задает включение в роспись текстовых заметок о биографии человека.
  • Настройка "Включая источники" - задает необходимость пометки каждого человека номерами источников, в которых он упоминается. Список источников приводится в конце росписи.
  • Настройка "Формат": "Избыточный" или "Традиционный", определяет насколько детально будет описан человек в росписи. В большинстве случаев предпочительно использовать традиционный формат, т.к. избыточный содержит много лишней информации.
