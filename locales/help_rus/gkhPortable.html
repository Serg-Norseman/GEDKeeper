Переносимое приложение

Переносимое приложение — программное обеспечение, которое для своего запуска не требует процедуры установки и может полностью храниться на съёмных носителях информации, что позволяет использовать данное ПО на многих компьютерах. Переносимое приложение разработано так, чтобы считывать свои конфигурационные настройки во время запуска из того же места, где хранятся его собственные программные файлы.

Первый способ: чтобы сделать приложение GEDKeeper переносимым, просто создайте папку `appdata` (для Windows) в каталоге, где находится GEDKeeper2.exe.

Второй метод: «GEDKeeper» имеет параметр командной строки `-homedir:path`, в котором указывается абсолютный или относительный путь, где хранить файлы конфигурации приложения. Пользователь может определить относительный каталог, если он/она хочет указать «текущий каталог» (текущий для среды или для двоичного кода GEDKeeper). Если параметр не используется, «домашний каталог» по умолчанию имеет системный зависимый путь (`%APPDATA%\GEDKeeper2`).

В том случае, если в папке запуска программы находится подпапка `appdata` и задан аргумент командной строки `-homedir:path` - приоритет будет за аргументом командной строки и вложенная папка будет проигнорирована.
