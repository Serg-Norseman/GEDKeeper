Статистика

Окно статистики предназначено для предоставления различной сводной информации о людях, содержащихся в базе данных.

Окно разделено на три зоны: вверху выводится общая сводка по всем людям, нижний блок предназначен для статистики по отдельным специализированным категориям.

Нижний блок состоит из выпадающего списка с различными категориями статистики, ниже располагаются список с данными и внизу-справа - график для некоторых из категорий.

Если персональные записи отфильтрованы, статистика формируется только по отфильтрованному результату.

Результат любой отдельной категории статистики можно экспортировать в Excel (кнопка ).
