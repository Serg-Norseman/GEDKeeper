Языки интерфейса

Программа GEDKeeper имеет функцию переключения между языками интерфейса. Интерфейс может быть как русским, так и любым другим. Язык можно переключить в меню "Сервис\Настройка", закладка окна "Общее", при этом переключение языка происходит сразу, без перезагрузки программы.

На данный момент программа переведена на следующие языки:

  • Русский [rus]
  • Английский [enu]
  • Украинский [ukr]
  • Польский [pol]
  • Итальянский [ita]
  • Китайский (упрощенное письмо) [zho]
  • Французский [fra]
  • Немецкий [deu]
  • Чешский [cze]
  • Казахский (кириллица), частично [kaz]

Любой пользователь может внести посильный вклад в развитие программы, переведя интерфейс на свой родной язык. Для этого нужно перевести наиболее подходящий пользователю файл "lng" из папки программы "locales" и прислать перевод разработчикам программы. После получения перевода и проверки соответствия данных, он будет включен в следующую версию программы.
