Горячие клавиши

Сводная таблица горячих клавиш в программе:

Основное окно
[ Ctrl + N ] Создать новую базу данных
[ Ctrl + O ] Открыть существующую базу данных
[ Ctrl + X ] Выйти из программы
[ Ctrl + F11 ] Открыть окно скриптов
[ F1 ] Открыть справку (этот документ)
Окно базы данных, таблицы записей
[ Ctrl + S ] Сохранить базу данных
[ Ctrl + I ] Создать новую запись
[ Ctrl + Enter ] Редактировать запись
[ Ctrl + L ] Удалить запись
[ Ctrl + A ] Древо предков
[ Ctrl + D ] Древо потомков
[ Ctrl + P ] Родословна роспись по д'Абовиллю
[ Ctrl + K ] Родословна роспись по Коновалову
[ Ctrl + M ] Карты
[ Ctrl + T ] Статистика
[ Ctrl + F ] Фильтрация данных
Окно диаграммы дерева
[ F5 ] Полное обновление всего изображения древа
[ F6 ] Устанавливает текущую выделенную персону как точку отсчета и выполняет полное обновление отображаемых предков и потомков относительно новой персоны
[ F7 ] Полный пересчет степеней родства (если включено их отображение) относительно текущей выделенной персоны
