Записи задач

Записи задач предназначены для хранения информации по исследованию конкретного человека, семьи или определенного источника. Для этого в записях можно указать тип цели, саму запись человека или семьи, приоритет задачи и сроки выполнения. Также задачу можно снабдить текстовыми заметками о ходе выполнения поиска.

Записи задач группируются по направлениям поиска при помощи записей исследований.

Все имеющиеся в базе данных задачи доступны через единую таблицу на закладке "Задачи" основного рабочего окна. Справа от таблицы - в сводке по записи, можно просмотреть всю информацию, а также список всех записей исследований, которые ссылаются на данную задачу.

Также смотрите: Персоны, Семьи, Источники, Заметки, Исследования.
