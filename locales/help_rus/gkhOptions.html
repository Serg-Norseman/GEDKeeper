Интерфейс

Подсвечивание персон без родителей или без семьи

Подсвечивание персон без родителей или без семьи полезно для улучшенной навигации в списке - по цвету персоны в списке всегда сразу видно: белая строка - у персоны определены и родители, и семья; светло-желтая строка - не задана семья; светло-красная - не заданы родители. Очень светлые цвета сделаны для того, чтобы подсветка не мешала восприятию информации.

Родословные древа

Для вывода родословных деревьев используется две группы настроек: настройки отображения персон в древе и настройки оформления.

Отображение персон в древе

В этой группе настроек определяется состав отображаемых данных: нужно-ли выводить фамилии, имена и отчества, даты рождения и смерти, необходимость вывода только годов в датах.

При включении опции "Степень родства" при каждой перестройке древа для всех персон будет выводиться их степень родства по отношению к центральному человеку в древе.

Опция "Дополнительные символы" используется для вывода специальных символов для некоторых категорий людей: специальные иконки для участников ВОВ, погибших, ветеранов труда, кавалеров Георгиевского креста, репрессированных. Эти категории задаются при помощи раздела "Сноски" в редакторе персон.

Опция "Декоративное оформление" выключает отдельные графические элементы и тени.

Опция "Исключить умерших в детстве" полезна при активной работе с данными при обработке различных источников, чтобы сократить объем выводимых персон и упростить работу.

Оформление (цвета и шрифт)

Эта группа настроек определяет внешнее цветовое представление и основной шрифт вывода деревьев на экран.

Росписи

Здесь располагаются настройки генерации родословной росписи. Настройка "Включая атрибуты персон" - задаёт то, будут ли включаться в роспись факты из жизни человека. Настройка "Включая заметки" - задает включение в роспись текстовых заметок о биографии человека. Настройка "Включая источники" - задает необходимость пометки каждого человека номерами источников, в которых он упоминается. Список источников приводится в конце росписи. Настройка "Формат": "Избыточный" или "Традиционный", определяет насколько детально будет описан человек в росписи. В большинстве случаев предпочительно использовать традиционный формат, т.к. избыточный содержит много лишней информации.
