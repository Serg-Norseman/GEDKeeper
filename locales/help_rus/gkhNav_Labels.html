Метки индивидуальных записей

Программа GEDKeeper поддерживает два типа меток индивидуальных записей для облегчения навигации: метки патриархов и закладки. Фактически разницы между ними немного. Обе метки задаются флажками в диалоге редактирования персональной записи. Метки эти формальны и введены только для удобства навигации по большим базам данных.

Меткой патриарха лучше всего отмечать наиболее глубокого предка мужского пола в каждой исследуемой линии - основателей фамилий.

Закладки служат преимущественно для пометки людей, по которым ведется наиболее интенсивный поиск в текущее время.

Когда метки установлены, их можно использовать двумя способами:

  • Метки патриархов и закладки отображаются в отдельных столбцах основного списка персон, поэтому можно легко отсортировать весь список так, чтобы все отмеченные люди оказались вверху списка, после чего можно сразу работать с одним из них;
  • Метки патриархов также доступны в диалоге фильтрации персональных записей, благодаря чему можно очень быстро получить короткий список всех предков - основателей линий, и сразу найдя нужного - вывести древо его потомков или построить родословную роспись.
