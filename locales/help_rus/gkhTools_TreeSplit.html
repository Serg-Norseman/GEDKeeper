Инструменты: Разделить базу данных

Разделение базы данных используется в тех случаях, когда необходимо либо полностью удалить каких-то людей или семьи из базы данных, либо сделать их копию для переноса в другую базу данных, либо сделать их копию для передачи другим генеалогам.

До вызова данного инструмента необходимо выбрать любого человека из будущей выборки в основном списке персональных записей (закладка "Персоны"). Затем, в инструменте есть четыре способа выбора нужных людей: выбрать все взаимосвязи, выбрать только предков, выбрать только потомков или только семью. Когда все необходимые персоны выбраны, можно воспользоваться сохранением их в отдельный файл базы данных или удалить. Операции можно совмещать, т.е. сначала сохранить выбранное в отдельный файл, а затем удалить из текущего.

В интерфейсе этого инструмента, список слева содержит всех выбранных людей, список справа - всех не вошедших в выборку.
