События

События и факты - это один из важнейших типов записей в генеалогической базе данных. События можно указывать только в персональных и семейных записях, где они служат для указания любых вех в жизни человека или семьи.

Для любого события можно указать основной тип, но если подходящего в списке не найдено, тогда можно указать обобщенный тип "Факт" и в текстовом поле справа указать своё название. Также для события можно указать место, где оно случилось и дату или диапазон дат, включая и календарь по которому нужно отсчитывать время. Кроме этой основной информации, события можно снабжать текстовыми заметками, ссылками на источники и мультимедиа-материалы. Для некоторых событий имеет смысл указывать причину (например, смерти) и засвидетельствующую организацию.

Поле "Значение" используется только для событий типа "Факт". Например для факта "Звание" в этом поле можно указать звание в котором человек служил в армии. Для факта "Награда", можно указать вид полученной награды. Программа в процессе работы постоянно собирает данные, которые были введены в поле "Значение" фактов, поэтому из выпадающего списка можно выбирать уже ранее введенные.

Статья Особенности работы с датами раскрывает все основные тонкости обработки дат в программе.
