Ancestors Circle

The "Ancestors Circle" is a special-purpose diagram that shows all ancestors of the selected person as sectors of concentric circles. Settings for this diagram is available at "Charts\Ancestors Circle" tab in the "Options" window. You can open the "Options" dialog using the "Service\Options..." menu. Settings on the specified tab allows you to set up the diagram appearance.

To generate an "Ancestors Circle" diagram, select a target entry in the person list and then use the "Ancestors Circle" item in the "Pedigree" menu.

The following actions are available while you are viewing the diagram:

  • The [Left] key updates the diagram for the father of the selected person.
  • The [Right] key updates the diagram for the mother of the selected person.
  • The [Up] key increases the number of displayed generations.
  • The [Down] key decreases the number of displayed generations.
  • The [Backspace] key returns to the previous person.
  • Click a sector to update the diagram of the selected person.
  • Double-click outside the diagram to change the current view mode to ancestors compositional analysis mode or revert the current mode back.

The diagram always shows a fixed number of ancestral generations (eight generations).

If you have changed the central person by clicking a sector of the diagram, you can use and buttons to switch to a previous or next person.

There is an additional ancestors compositional analysis mode available in the "Ancestors circle" diagram: you can double-click outside the diagram to conduct an ancestor duplication analysis. This analysis changes the diagram as follow: all unique, non-duplicated ancestors have chlorine color, while all ancestors having two or more occurrence have a different color (each duplicating ancestor has its own color). This allows you to evaluate genetic variety of ancestors more clearly.
