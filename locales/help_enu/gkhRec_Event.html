Event

There are two main record types in a genealogical database: events and facts. An event designates a milestone in person or family life. An event can only be attached to a person or family record.

Each event has a type—it can be a predefined one or you can use a "Fact" type as a generic type and then add a user-defined type name. For any event a location where it took place may be specified as well as a date or date range with a calendar. Optionally you can add some text notes, reference to sources, and multimedia content. For some events like death it is not unreasonable to specify a reason that caused the event and a certified organization.

The "Value" field is valid only for events of type "Fact". For example, the "Value" field for a "Rank" fact may store the rank a person had while performing military service. An "Award" may store any kind of award (e.g., the purple heart). All values you have entered are stored for quick access to any previous value (from the dropdown list).

There is more information about how you can define and process dates in the Dates Special Features section.
