Keyboard Shortcuts

The following is a list of hotkeys available in GEDKeeper:

Main window
[ Ctrl + N ] Create a new database
[ Ctrl + O ] Open an existing database
[ Ctrl + X ] Exit the application
[ Ctrl + F11 ] Open the scripts window
[ F1 ] Show help content (this document)
Database window, records tables
[ Ctrl + S ] Saves the current database
[ Ctrl + I ] Adds a new record
[ Ctrl + Enter ] Modifies a record
[ Ctrl + L ] Deletes a record
[ Ctrl + A ] View an ancestors tree
[ Ctrl + D ] View a descendants tree
[ Ctrl + P ] Create a pedigree according to the d'Aboville scheme
[ Ctrl + K ] Create a pedigree according to the Konovalov scheme
[ Ctrl + M ] Open the maps
[ Ctrl + T ] Open statistics
[ Ctrl + F ] Filter data
Tree diagram window
[ F5 ] Rebuild the current tree
[ F6 ] Make the selected person record a starting point and rebuild the
[ F7 ] When showing of relation degrees is enabled: rebuild relation degrees for the selected person
