Keyboard Shortcuts

The following is the list of hotkeys available in GEDKeeper:

Main window
[Ctrl + N] Create a new database.
[Ctrl + O] Open an existing database.
[Ctrl + X] Exit the application.
[Ctrl + F11] Open the scripts window
[F1] Show help content (this document).
Database window, records tables
[Ctrl + S] Save the current database.
[Ctrl + I] Add a new record.
[Ctrl + Enter] Modify the selected record.
[Ctrl + L] Delete the selected record.
[Ctrl + A] View the ancestors tree.
[Ctrl + D] View the descendants tree.
[Ctrl + P] Create a pedigree according to the d'Aboville scheme.
[Ctrl + K] Create a pedigree according to the Konovalov scheme.
[Ctrl + M] Open maps
[Ctrl + T] Open statistics
[Ctrl + F] Filter data
Tree diagram window
[F5] Rebuild the tree.
[F6] Make selected person record as a start point and rebuild ancestors and descendants of the selected person.
[F7] When showing of relation degrees is enabled, this will rebuild relation degrees for the selected person.
