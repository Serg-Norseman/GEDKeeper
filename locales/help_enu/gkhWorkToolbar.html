Main Toolbar

GEDKeeper's main toolbar is shown on the following picture:

The toolbar has the following buttons:

Database management
 Create a new database.
 Open an existing database.
 Save all database modifications.
Record management (depending on the active tab)
 Add a new record.
 Modify the selected record.
 Remove the selected record.
 Filter a record list.
Family tree and pedigree ("Persons" tab only)
 Open an ancestor tree for the selected individual.
 Open a descendant tree for the selected individual.
 Open a combined tree with ancestors and descendants for the selected individual.
 Create a pedigree for the selected person.
Miscellaneous
 Show statistic information for individuals.
Navigation
 Move backward to the previous record.
 Move forward to the next record.
