Standard plugins

There are some plugin modules available with the default installation:

  • Expression calculator, a standard service plugin.
  • Calendar, a standard service plugin.
  • Names reference book, a standard service plugin.
  • Time line, a standard service plugin.
  • Streamed data input plugin streamlines mass data input from sources.
  • Pedigree import plugin adds data from an external text pedigree.
  • Image viewer plugin shows a sources' scanned images and photos.
  • Full-text search plugin searches for an entry in the database—works alike a Web search engine.
  • 3D Visualization (experimental) plugin shows 3D tree diagrams.
  • "Conway Life" game plugin helps you to relax.
