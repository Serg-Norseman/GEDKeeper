Standard Plugins

These are some of the plugin modules available with the default installation:

  • "Calc" is an expression calculator.
  • "Calendar" displays any date as one of several formats (Gregorian, Jullian, etc.).
  • "Names book" provides a reference of names within the database.
  • Time line filters the person lsit by year using a slider.
  • The "Stream input" plugin streamlines mass data input from sources.
  • The "Import pedigrees" plugin adds data from an external text pedigree.
  • "Image viewer" is a plugin that displays image (photo) sources.
  • The "Full-text search" plugin searches for an entry in the database—works like a Web search engine.
  • "3D Visualizer" is an experimental plugin to display 3D tree diagrams.
  • "Conway Life" game plugin is for a bit of relaxation.
