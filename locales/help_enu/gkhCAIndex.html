Authenticity Index

The Authenticity Index is an attribute of data about a person as a special indicator that can be evaluated for a person entity only.

Each reference to a source in a person entity has a specific authenticity level of mention for that person in the source. Authenticity level can have the following values:

  • Unreliable evidence or estimated data (authenticity is 25%).
  • Questionable reliability of evidence (authenticity is 50%).
  • Secondary evidence, data officially recorded sometime after event (authenticity is 75%).
  • Direct and primary evidences, or dominance of evidences (authenticity is 100%).

While you are researching and viewing some information, the authenticity level shows you how much you can trust that information. It is better to set "Questionable reliability of evidence" if a source document had blots or was damaged and could not be read with confidence. This can prevent you from making false conclusions.

If a reference to a source has no authenticity level specified in the database, it is interpreted by the program as "Unreliable evidence or estimated data" when calculating the Authenticity Index.

The Authenticity Index is evaluated automatically for each person based on authenticity level (as the weighted arithmetic mean). You can view the Authenticity Index in a statistics module for analysis of mean values or in tree diagrams for pictorial presentation. The index is evaluated using all references to sources, specified in both person entity and all materials and events.
