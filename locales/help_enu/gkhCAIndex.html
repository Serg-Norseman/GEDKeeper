Authenticity Index

The authenticity index ("certainty") of information about a person is a special indicator that can be evaluated for person entity only.

Each reference to a source in a person entity has a specific authenticity level when mentioning that person in the source. The authenticity level can have the following values:

  • "Unreliable evidence" indicates a 25% authenticity level.
  • "Questionable reliability of evidence" indicates a 50% level of authenticity.
  • "Circumstantial evidence" is data officially recorded sometime after event (authenticity is 75%).
  • "Direct and primary evidence" is considered a dominance of evidences (authenticity is 100%).

While researching and viewing some information, the authenticity level shows how much the information can be trusted. It is better to set "Questionable reliability of evidence" if a source document had blots or was damaged and could not be read with confidence. This can help prevent false conclusions.

If a reference to a source has no authenticity level, the reference is considered "Unreliable evidence or estimated data" by default. The mentioning of a person in a document is an important acknowledgement by itself.

The authenticity index is evaluated automatically for each person based on authenticity level (as a weighted arithmetic mean). You can view the authenticity index in statistics module for analysis of mean values or in tree diagrams for pictorial presentation. The index is evaluated using all references to sources—specified in both the person entity and all materials and the associated events.
