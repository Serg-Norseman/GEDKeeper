Archive Record

Archive records combine sources located in one storage place. An archive is either a state archive, a private archive institution, a private collection, or some specific document storage.

An archive has a name, storage unit list, an address, and other contact information. You can also add text notes to document items like hours of service for an instution.

There is a uniting table on the "Archives" tab of the main working window which includes all archives available in the database. An archive summary to the right of the table shows all information about the selected archive and sources that, according to the database, are stored in the selected archive.

See also: Source, Note, Address.
