Getting started

To get started working with GEDKeeper, you must first create an empty database by clicking the button on the toolbar or selecting "File\New" from the menu.

Using the button will create a new person record. This could be the central one for your family tree.

You can leave any field empty if you do not have the necessary information. As time goes by, you add new data and relations, thus forming your family tree.

If you need to load existing data, created by GEDKeeper or any other application compatible with the GEDCOM specification, click the button or select the "File/Open..." menu item. After you have selected a file, GEDKeeper loads it and you can start editing or processing the data. If you need to keep the original data intact, use the "File\Save As" menu item to save the data as a new database.
