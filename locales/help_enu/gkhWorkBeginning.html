Getting Started

When working with GEDKeeper you must first create an empty database. This is done by clicking the button on the toolbar.

Using the button creates a new person record. This is fundamental to creating a family tree.

You can leave any field empty if you currently do not the necessary information. As you continue to use GEDKeeper and add new data and relations you are forming a family tree.

If you need to load existing data (created by GEDKeeper or any other application compatible with the GEDCOM specification), click the "Open tree file" () button or click the "File/Open..." menu item. After a file is selected GEDKeeper will load it then you can start editing or process new data.
