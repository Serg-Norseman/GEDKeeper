Portable Application

A portable application is a program designed to read and write its configuration settings into an accessible folder in the computer. This generally refers to the folder where the portable application can be found. This makes it easy to transfer the program, with user preferences and data, to a different computer. There are two options for making GEDKeeper portable:

  • Copy the file GEDKeeper2.exe to a directory (e.g., 'c:\apps\GEDKeeper'). Then create a subdirectory for the application data where GEDKeeper2.exe is located (e.g., 'c:\apps\GEDKeeper\appddata').
  • GEDKeeper supports the command line option '-homedir:path' where path is the directory to store the configuration files (the settings). Both absolute and relative directories are supported. If this option is not used, the directory defaults to the system dependent path for a users' data (i.e., in Windows this would be '%APPDATA%\GEDKeeper2').

    If a subdirectory called 'appdata' exists in the directory where GEDKeeper is located then the '-homedir:path' option is ignonred.
