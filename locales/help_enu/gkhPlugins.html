Plug-ins

There are two types of extensions available for GEDKeeper: Lua scripts and plug-ins written in C# or VB.NET.

The following are the main reasons to extend an application with plug-in units:

  • You want to add new functions to GEDKeeper without having to rebuild the source code and deploy a new release.
  • You want to add new functions to the application but do not have the required source code (which is freely available).

To create plug-ins you must use the GKCore common library of this application's core.
