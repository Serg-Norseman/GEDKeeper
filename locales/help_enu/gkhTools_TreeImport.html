Tools: Pedigrees Import

Attention:

  • An import processes pedigrees in text format only.
  • A person must be identified by Konovalov's scheme.
  • A person identifier and its name must be delimited by a period (for example, "2-1. Daniel Garner").
  • The name must have the following format: "First_name Patronymic_name Last_name".
  • The sex of the person is determined by typical features of the Russian language's woman names, on names reference index, and, in all other cases, on user decision (via prompting).

A recommendation: perform several imports and check the results after each one. If you find an error, modify the original text. For example, frequently birthdays inside a row with a name are not marked with the "*" symbol. It is impossible to recognize such situations programmatically; this may increase the number of false positives. Moreover, the name may contain name variants enclosed in parentheses, without parentheses, or with the "?" indicator. It is impossible to recognize these situations by GEDKeeper. It is better to extract such data, and various date formats, to a text block after the row with the name.
