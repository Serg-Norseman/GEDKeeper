Tools: Import pedigrees

Characteristics of the input file:

  • Import processes pedigrees in text format only.
  • Each person must be identified by the Konovalov scheme.
  • The person identifier and its name must be delimited by a period (for example, "2-1. Daniel Garner").
  • Name must have the following format: "First_name Patronymic_name Last_name".
  • The person's sex is determined based on typical features of the Russian language's woman names on the names reference index, and in all other cases on responding to a prompt.

A recommendation: Create a temporary database and make several import attempts and check the results after each attempt. If you find an error, modify the original text. For example, frequently birthdays inside a row with a name are not marked with the "*" symbol. It is impossible to recognize such situations programmatically—this would increase the number of false positives. Moreover, a name may contain name variants enclosed in parentheses, without parentheses, or with the "?" symbol. It is impossible to recognize these with a simple algorithm. It is better to extract such data, and various date formats, to a text block after the row with a name.
