Source Record

A source is referencing any document or file from the archive collection where information about a person is found. Source records are used for the following purposes: storing specific information about the origin of the information and to unite and simplify a search for person, family, and event that were mentioned in the document.

This can be a birth or death certificate, audit tale, register of births, confessional book, census report, and other documents.

Each source reference has two additional properties: page or sheet number where the mentioning was made, and authenticity level for the information. Having a page or sheet number is helpful when you need to review the source once again. Authenticity level quickly shows you how much you can trust a source of information. When a source document cannot be viewed with assurance of its' reliability due to blots or other damage, we recommend assigning the "Questionable reliability of evidence" level to the source. This can help prevent false conclusions.

Authenticity level also affects the authenticity index; which is calculated for a person in the statistics module to analyze mean values. Tree diagrams use authenticity levels to make displayed information more obvious.

Source also has an archive property that defines where the source is stored. It is worth noting that the archive here is not necessarily a state archive or private archive institution, but a private document collection or other specific document storage.

There is a uniting table on the "Sources" tab of the main working window which includes all sources available in the database. A source summary to the right of the table shows all information about the selected source and links made from other records to this one.
