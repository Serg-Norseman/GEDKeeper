Note Record

Notes are used to add minimal, unformatted text to most record types. A note can contain fragments of a source text, scratches and drafts added to persons, families or multimedia content, research notes, iniformation about location history, and so on.

Please do not take a note object as some kind of large formatted document. The latter must be added to the database as multimedia content. The number of notes is unlimited—as is references made to a note from other objects.

There is a table on the "Notes" tab of the main working window where you can find all notes stored in the database. Each note record has a summary area to the right of the table where you can find the list of notes that refer to the selected note.
