Pedigrees

GEDKeeper has the ability to generate a generational pedigree. There are two pedigree schemes available: d'Aboville's and Konovalov's. The only difference betweeen the two is the person numbering method.

d'Aboville's scheme assigns #1 to the first person in the first generation. The person's first descendant in the next generation is #1.1, the grandchild is #1.1.1, and so on. This system is suitable when you have started to create a tree draft on paper—it is easy to add elements to any generation in the pedigree using d'Aboville's scheme.

Konovalov's scheme gives numbers like "X-Y" to all people, where "X" is a sequential number for a person in the pedigree, and "Y" is the number of the person's ancestor. This system provides easy navigation but may be hard to manage on paper. Adding new persons to existing generations would be a tough task; numbering get broken and the pedigree becomes a mess.

GEDKeeper sees little difference between the two schemes. Choosing a scheme is based on your personal preference (appearance of pedigree, and how to view and manage a pedigree).

To create a pedigree using one of the schemes described above, select an entry in the person list. Then find the menu item with the scheme name in the "Pedigree" menu. Alternately you can use the button on the toolbar.

You can export pedigrees to HTML, PDF, and RTF formats. The exported file includes hyperlinks between persons in the tree. Once GEDKeeper has created the target file, it then opens it with the registered program in your system as the default action for the target file type. You can use that program to print or modify the pedigree.

You can find settings for pedigree generation in the "Options" dialog on the "Pedigrees" tab including:

  • When "Include person attributes" option is on, pedigree includes a person's life events.
  • When "Include notes" option is on, pedigree includes text notes from a person's biography.
  • When "Include sources" option is on, pedigree includes all source numbers that references a person. There is a list of sources at the bottom of a pedigree.
  • The "Format" option has two possible values: "Excess" and "Compact". This option controls the level of detail for person description printed in the pedigree. Using the Compact format is generally used since the excess one may contain too much unnecessary information.
