Localization

GEDKeeper has multi-language support. You can use, for example, the Russian localization, or another one, at your preference. You can change the user interface localization by using "Service\Options" on the "Common" tab. The interface language is changed immediately—restarting the application is unnecessary.

You can contribute to the project by translating the user interface into your native language. All it takes is copying the most appropriate ".lng" file from the "locales" project subdirectory and email the translated file to the author of GEDKeeper. After validation, your file will be included in the next GEDKeeper release.
