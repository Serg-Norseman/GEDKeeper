Localization

GEDKeeper has multilanguage support. You can use, for example, the Russian localization or another one at your option. You can change the user interface localization using "Service\Options" on the "Common" tab. The interface language is changed immediately—you do not have to restart the application.

Also, you can contribute to the project by translating the user interface into your native language. Just translate the most suitable "lng" file from the "locales" project subdirectory and email the translated file to the author of GEDKeeper. After validation, your file will be included in the next GEDKeeper release.
