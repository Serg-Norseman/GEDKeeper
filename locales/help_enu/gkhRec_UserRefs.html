User Reference

A user reference/footnote is a special data type used in person records to add miscellaneous notes. The GEDCOM standard does not define a purpose for this object—it was introduced only for user comfort, to be used as short note.

However, GEDKeeper defines the purpose for user references: to show special marks on persons in tree diagrams. For example, if a person waged The Great Patriotic War and his/her person record has the respective reference, then this person is shown in tree diagrams with a red star. If a person died, a respective reference adds a black edge to the star.

The following predefined user references are available in GEDKeeper:

 USSR:WW II:Rear worker
 USSR:WW II:Combatant
 USSR:WW II:Killed in action
 RE:George knight
 USSR:Victimizated
 Religion:Islam
 Religion:Catholicism
 Religion:Orthodoxy
 Religion:Old Belief

Additional predefined user references may be added to GEDKeeper in future releases. Of course you can always create your own references.

You change user references on the "Reference/footnote" tab of the person editor window.
