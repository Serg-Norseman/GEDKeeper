Family Tree

Three types of tree diagrams are available: tree of ancestors (ascending relationship tree), tree of descendants (descending relationship tree), and combined tree (ancestors and descendants). You can set up the diagram appearance in the "Options" dialog on the "Tree" tab (accessed by the "Options" dialog via the "Edit/Options..." menu item). You can change information to show about persons and change colors of tree elements.

To build a tree, you pick a person from the person list. Then choose a tree type (ancestors, descendants, or complete) in the "Pedigree" menu. The following main window's toolbar's buttons may also be used:

  •  Tree of ancestors
  •  Tree of descendants
  •  Complete tree

After a tree is created, you can use the button to export the tree in one of the following formats: raster format (BMP, JPEG, PNG or, GIF) or vector format (EMF, SVG). Not every family tree may be saved as a raster (bitmap) file. Bitmaps may not store more than 65k pixels per horizontal line. Vector files do not have such restriction; therefore it can store trees of any size. Moreover, a vector file is better for additional image processing in graphical applications and for printing.

The following controls are available when you view a diagram:

  • Vertical scrolling with mouse wheel.
  • Horizontal scrolling with mouse wheel and holding the [ Shift ] key.
  • Zoom with mouse wheel and holding the [ Ctrl ] key.
  • Zoom with scale control at top right corner of the window.
  • [ F4 ] performs the folding or unfolding of the selected person's branch.
  • [ F5 ] rebuilds the tree.
  • [ F6 ] marks the currently selected person as a starting point and updates all ancestors and descendants of the starting point.
  • [ F7 ] recalculates the relation degrees of the selected person (if that option is turned on).
  • Double-click on a person to open it in edit mode.
  • Click on a person to adjust it with the window's center.
  • Pan the content while holding the right mouse button.
  • Click on the icon ("if there is additional hidden data") near a person to rebuild the tree relative to that person (this icon is not exported to any image file).

The number of visible generations, of both ancestors and descendants, can be adjusted from one to nine or unlimited by the button with drop down menu on the window's toolbar.

Content may also be filtered by the button.

If the [ F6 ] key is used or the icon (set new person as a central one for the tree), then you can use the and buttons to return to the previous or next person.

The icon dispalys the following tree options:

  • All -- show both ancestors and descendants for a selected person.
  • Ancestors only -- only ancestors of the selected person are visible.
  • Descendants only -- only descendants of the selected person are visible.
  • Center auto-change -- toggle off/on a mode when the center of the tree is smoothly adjusted with the selected person.
  • Authenticity index -- toggle off/on the authenticity index visibility for each person in the tree (it is handy for visual valuation of development progress).
  • Background color -- change the background color for this tree.
  • Background image -- change the background image for this tree.
