Tools: Check Family Connectivity

This is an analysis tool—it is used to check the number of fully interrelated families. The tool starts from a person and then processes all persons that have any relation with the initial person—irrespective of relation degree. All inspected persons are combined into one group. All persons interrelated between each other, but not with the first group, are combined in other groups.

You can develop several unrelated families in one database. This tool provides the ability to visually manage their composition. Also, you can use this tool as an integrity checking procedure. For example, in cases when you removed a person from a family but did not add that person to a different family. In such cases, this tool displayes the person as disjointed from all groups.

Double-click on a person in a group list to open this person in the main working list of the database. This may be helpful when you want to make some changes to a person.

There is a database fragmentation diagram under the list of group contents. For better visual perception, the diagram has a logarithmic scale. Therefore the size of a piece is not directly proportional to the number of persons in the relative group. This decreases the size of large segments, but keeps small fragments clearly visible. Hovering the mouse over a fragment shows you the fragment/group number and its actual size.
