Group record

Groups help with navigation—they group people by a common social identifier.

For example, you can create a "Participants in The Great Patriotic War" group with respective members. Or create a "Correspondents" group for people you are currently staying in contact with.

To add a new person to a group you can use any of the following methods:

  • Click the "Add" button () on the "Members" tab of the group editing dialog.
  • Click the "Add" button () on the "Groups" tab of the group editing dialog.

Using the "Remove" button () from the same tabs, you can remove persons from groups.

There is a table on the "Groups" tab of the main working window where you can find all groups stored in the database. Each group record has a summary area to the right of the table where you can view the entire information about a group and a full list of members.
