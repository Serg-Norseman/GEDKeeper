Task Record

A task record stores information about investigation into a specific person, family, or source. For this purpose, a task has a goal, a reference to person or family, a task priority, and due date. Text notes may be added to a task to describe search progress.

You group task by search direction using research records.

There is a uniting table on the "Tasks" tab of the main working window which includes all tasks available in the database. A task summary to the right of the table shows all information about the selected task, and the list of research records that link to the selected one.

See also: Person, Family, Source, Note, Research.
