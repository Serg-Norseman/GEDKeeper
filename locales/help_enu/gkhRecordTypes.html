Types of Database Records

GEDKeeper stores database records of the following types:

  • Person
  • Family
  • Note (text attached to a database record, piece of document, or comment)
  • Multimedia content (reference to a document, photo, audio, or video recording)
  • Source (a private document or a document from an archive fund)
  • Archive (state archive and private archive)
  • Group (group of people having common social attributes)
  • Research (main directions of investigation and priority of those directions)
  • Task (discrete investigation task)
  • Correspondence (information about correspondence, communication with relatives, and other researchers and archive organizations)
  • Location (geographical location and location coordinates)

The main working window has a dedicated tab for each record type:

The following operations are available for all record types: add, modify, remove, and filter.

Each tab has two visual parts: a table with a record list on the left and a summary window that displays detailed information about the selected entry in the table.

Entries in the table may be ordered by clicking a column—a property of each record. To avoid incidental changes, you cannot change data directly in the table (in-place editing is disabled). To open an editor window for a record you can double-click the row or select a row and press the [ Ctrl + Enter ] key combination.

All record types have two common attributes: record identifier and last modification time ("#" and "Changed" columns respectively). All other attributes in a record are record type specific.

You can activate a filter on the table using the button on the toolbar, using the "Edit\Filter" menu item, or pressing the [ Ctrl + F ] hotkey. In addition, the "Persons" tab has a quick search feature from the "Edit\Search" menu item.
