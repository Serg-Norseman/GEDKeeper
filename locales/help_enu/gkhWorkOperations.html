File operations

Create new file

Select the "New" menu item in the "File" menu or click the button on the toolbar.

You may also use the [Ctrl + N] hotkey.

Open existing file

Select the "Open..." menu item in the "File" menu or click the button on the toolbar.

You may also use the [Ctrl + O] hotkey.

Save file

Select the "Save..." menu item in the "File" menu or click the button on the toolbar.

You may also use the [Ctrl + S] hotkey.

File properties

Select the "File properties" menu item in the "File" menu. Using the opened window, specify a document author. In this window you can view file summary information and a file integrity report.

Record operation

View record

Find and select a record you need in a list. You can view a complete list of record properties to the right of record list. Summary information may contain hyperlinks that assist in your navigation to other records.

Add record

Click the tab with the required record type (i.e., Individuals, Families, Notes, etc.). Then click the button, or use the [Ctrl + I] hotkey, to open an editor window. Add everything you know about the object then press the "Apply" button to save the data. This new record will now display in the list.

For example, click the "Individuals" tab. Now click the button. The person editing window appears where you can add the information about that person. Once done, press the "Apply" button to save your data and close the window.

Modify record

Find a record you need on a tab with a record list. Select the "Modify record" menu item in the "Edit" menu or click the button.

Every record type has its own editor window:

  • Person editor
  • Family editor
  • Note editor
  • Multimedia content reference editor
  • Source editor
  • Archive editor
  • Group editor
  • Researching editor
  • Task editor
  • Correspondence editor
  • Location editor

Remove record

Find a necessary record on a tab, select it and use the "Remove record" menu item in the "Edit" menu, or click the button, or use the [Ctrl + L] hotkey.
