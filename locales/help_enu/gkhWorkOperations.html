File Operations

Create a New File

Select the "New" menu item in the "File" menu or click the button on the toolbar.

Open Existing File

Select the "Open..." menu item in the "File" menu or click the button on the toolbar.

Save File

Select the "Save..." menu item in the "File" menu or click the button on the toolbar.

File Properties

Select the "File properties" menu item the in "File" menu. Using the opened window you specify the document author. In this window you can view file summary information and a file integrity checking report.

Record Operation

View Record

Find and select a record you need in a list. You can view a complete list of record properties to the right of the record list. Summary information may contain hyperlinks that make your navigation through records easy.

Add Record

Havie a tab with a list of the necessary record type open. This is done by clicking "Persons", then selecting the "Add record" menu item in the "Edit" menu or clicking the button. A "new person" window is opened (or whatever is appropriate for the record type). This is where you specify everything you know about the object. When done, press the "Apply" button to save your data. A new entry will appear in the record list.

Modify Record

Find a record you need on a tab on record list. Select the "Modify record" menu item in the "Edit" menu or click the button.

Every record type has its own editor window:

  • Person editor
  • Family editor
  • Note editor
  • Multimedia content reference editor
  • Source editor
  • Archive editor
  • Group editor
  • Researching editor
  • Task editor
  • Correspondence editor
  • Location editor

Remove Record

Find the desired record from a list on a tab and select it. Then use the "Delete record" menu item in the "Edit" menu or click the button.
