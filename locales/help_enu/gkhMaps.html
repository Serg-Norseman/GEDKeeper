Maps

GEDKeeper uses Google Maps (© Google) to display maps.

Using the maps service allows GEDKeeper to show you locations of births, deaths and the living, as defined in the database, right on a map. There are two modes available: one shows births, deaths or living location for all persons in the database; the other shows all locations of any event for the selected person.

Before utilizing the map service, you can filter persons out using the general GEDKeeper filters.

Currently printing of the maps is not supported.
