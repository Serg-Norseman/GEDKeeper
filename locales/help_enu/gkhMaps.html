Maps

GEDKeeper uses Google Maps (© Google) to show maps.

Using a map service allows GEDKeeper to show you locations of births, deaths, and living, defined in database, right on a map. There are two modes available: one shows births, deaths, or living location for all persons in database; the other one shows all locations of any event for the selected person.

Before calling the map service, you can filter persons using the general GEDKeeper filters.
