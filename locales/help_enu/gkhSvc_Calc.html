Calculator

GEDKeeper has an easy-to-use calculator that allows you to evaluate expressions using common mathematical operations and temporary variables.

This calculator is very useful when bulk information sources must be processed. For example, with audit tales, confessional books, or census reports.

You can work with the calculator and other windows simultaneously. Therefore, for example, you can drag a calculation result and drop it into a date field inside the event editor dialog. Thus, while you are processing a source, you can enter an expression of "1896-56" into the calculator (a year in a source "-" person age "=" year of birth), and then drag the result into a date field (the result is treated as year).
