Инструменты: Разделить базу данных

Разделение базы данных используется в тех случаях, когда необходимо либо полностью исключить какие-то лица или семьи из базы данных, либо сделать их копию для переноса в другую базу данных или для передачи иным генеалогам.

До вызова данного инструмента необходимо выбрать в основном списке персональных записей, необходимого человека. Затем, в инструменте есть четыре способа выбора нужных персон: выбрать все взаимосвязи, выбрать только предков, выбрать только потомков или только семью. Когда все необходимые персоны выбраны, можно воспользоваться сохранением их в отдельный файл базы данных или удалить (операции можно совмещать, т.е. сначала сохранить выбранное в отдельный файл, а затем удалить из текущего).
