Инструменты: Объединить базы данных

Данный инструмент в настоящее время может использоваться только для прямого объединения двух баз данных. Все записи из одной базы полностью переливаются в другую. Сами записи при этом не модифицируются и не объединяются между собой, взаимосвязи также не меняются.

После объединения баз данных полезно применить инструмент для объединения дубликатов записей - в том случае, если в обеих базах были одни и те-же лица.
